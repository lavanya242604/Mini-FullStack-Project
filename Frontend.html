HTML & CSS: The Backbone of the Web

HTML (HyperText Markup Language) is the standard language used to create and structure content on the web. CSS (Cascading Style Sheets) is used to control the presentation, formatting, and layout of HTML elements. Together, they form the foundation of web development.

History of HTML & CSS:

HTML was developed by Tim Berners-Lee in 1991 to help share scientific documents online. Since then, it has evolved through several versions, with HTML5 being the latest. CSS was introduced in 1996 by the W3C to separate content from design, allowing for cleaner and more manageable code.

Applications of HTML & CSS:

  • Web Pages: The most basic use is creating and styling websites.
  • Email Templates: Used in designing visually appealing email layouts.
  • Web Applications: Front-end interfaces are built using HTML and CSS.
  • Online Forms: Collecting user data via styled form elements.
  • Digital Resumes & Portfolios: Personal branding in a web-friendly format.
  • Prototyping: Designers use HTML/CSS to build static prototypes.

Common HTML & CSS Tags:

  • <h1> to <h6>: Heading tags used for titles and subtitles.
  • <p>: Paragraph tag used to define blocks of text.
  • <a>: Anchor tag used to create hyperlinks.
  • <img>: Used to embed images in a web page.
  • <div>: Container tag used to group content for styling with CSS.
  • class & id: HTML attributes used with CSS for styling specific elements.
  • color, margin, padding: CSS properties used to style content.
  • display: Defines the layout behavior (block, inline, flex, etc.).

Examples:

a) HTML Basic Page Structure:
<!DOCTYPE html>
<html>
  <head>
    <title>My First Page</title>
  </head>
  <body>
    <h1>Welcome!</h1>
    <p>This is my first webpage.</p>
  </body>
</html>

b) CSS Styling Example:
<style>
body {
  background-color: lightgray;
  font-family: Arial;
}
h1 {
  color: blue;
}
</style>
